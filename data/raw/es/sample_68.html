Bajas

[editar]

Las siguientes parejas se bajaron del torneo antes de su inicio:

  • Amanda Anisimova / Markéta Vondroušová
  • Tímea Babos / Kristina Mladenovic
  • Alizé Cornet / Heather Watson
  • Vera Lapko / Dayana Yastremska
  • Magda Linette / Bernarda Pera
  • Květa Peschke / Rosalie van der Hoek

Wild cards

[editar]

Las siguientes parejas recibieron invitaciones para participar en el cuadro principal (WC):

  • Destanee Aiava / Astra Sharma
  • Kimberly Birrell / Jaimee Fourlis
  • Lizette Cabrera / Maddison Inglis
  • Olivia Gadecki / Belinda Woolcock
  • Daria Gavrilova / Ellen Perez
  • Simona Halep / Charlotte Kempenaers-Pocz
  • Abbie Myers / Ivana Popovic

Ranking protegido

[editar]

Las siguientes parejas ingresaron al cuadro principal con ranking protegido (PR):

  • Mona Barthel / Lin Zhu
  • Kateryna Bondarenko / Nadiia Kichenok
  • Aleksandra Krunić / Martina Trevisan
  • Vera Lapko / Markéta Vondroušová
  • Elena Rybakina / Yaroslava Shvédova

Retiros

[editar]

Las siguientes parejas se retiraron durante el torneo (r):

  • Fiona Ferro / Jil Teichmann
  • Ashleigh Barty / Jennifer Brady

  • PR = Ranking protegido (Protected Ranking)
  • s = Partido suspendido (Suspended)
